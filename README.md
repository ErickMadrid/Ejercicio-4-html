# Ejercicio-4-html

Este ejercicio consiste en la creación de un sistema de navegación básico entre múltiples archivos locales.

## Estructura de archivos
*   `index.html`: Página principal con enlaces a las páginas secundarias.
*   `pagina2.html`: Página secundaria con enlace de retorno.
*   `pagina3.html`: Página secundaria con enlace de retorno.

## Etiquetas aplicadas
*   `<a>`: Utilizada con el atributo `href` apuntando a archivos locales para permitir la navegación entre páginas.
